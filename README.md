> 寻迹莫干山 · 记路家计划 — 48小时倒计时活动网页

## 项目结构

```
48Hproject/
├── invitation.html    ← 邀请函页面（首页，用户第一个看到的）
```

## 页面说明

### 1. invitation.html — 邀问函

用户的**入口页面**。展示活动基本信息（日期、地点、时长）和体验标签，不泄露具体行程。包含跳转到海报页面的按钮。

### 2. poster.html — 活动日程海报

核心页面，包含三个阶段的状态机：

| 阶段 | 说明 | 触发条件 |
|------|------|----------|
| **SECRET** | 倒计时保密页，显示距离解锁的倒计时 | 当前时间 < 2026-07-03 14:00 北京时间 |
| **LAUNCH** | 解锁后的启动页，点击「启动倒计时」按钮 | 倒计时归零后 |
| **RUNNING** | 运行中，显示实时倒计时 + 分日期解锁行程 | 点击启动按钮后 |

行程按日期自动解锁：
- DAY 01（7月3日）：14:00 解锁
- DAY 02（7月4日）：次日 00:00 解锁
- DAY 03（7月5日）：第三天 00:00 解锁

## ⚠️ 前端动效修改指南

### ✅ 可以修改的部分（视觉动效）

- **CSS 动画和过渡效果**：所有 `@keyframes`、`transition`、`animation` 属性
- **配色方案**：`:root` 中的 CSS 变量（`--orange`、`--lime`、`--teal` 等）
- **字体大小和间距**：`font-size`、`letter-spacing`、`padding`、`margin` 等
- **布局调整**：`flex`、`grid` 相关属性
- **响应式样式**：`@media` 查询中的样式
- **SVG 图形**：山峦轮廓、图标等 SVG 元素的视觉样式
- **悬停效果**：`:hover` 状态的视觉变化

### 🚫 不要修改的部分（内容和逻辑）

#### poster.html 中的 JavaScript 变量

```javascript
// ❌ 不要修改这些时间常量
const ANCHOR = new Date('2026-07-03T06:00:00Z');  // 活动开始时间（UTC）
const TOTAL_MS = 48 * 60 * 60 * 1000;              // 48小时毫秒数

// ❌ 不要修改日期解锁时间
const DAY_UNLOCK_UTC = {
  1: new Date('2026-07-03T06:00:00Z').getTime(),
  2: new Date('2026-07-03T16:00:00Z').getTime(),
  3: new Date('2026-07-04T16:00:00Z').getTime(),
};
```

#### poster.html 中的 HTML 内容

- **行程时间**：所有 `time-value` 中的时间（如 `14:00`、`15:00`）
- **行程内容**：所有 `event-name` 中的活动名称
- **T-MINUS 倒计时值**：`tminus-value` 和 `data-tminus` 属性
- **阶段标签**：`event-phase` 中的英文+中文标签
- **日期标签**：`day-label` 中的日期信息

#### 核心 HTML 结构

- `#secret-overlay` — 保密倒计时层
- `#launch-overlay` — 启动页层
- `#live-countdown` — 实时倒计时条
- `.day-section[data-day]` — 按日期分组的行程区块
- `.event-row[data-tminus]` — 每个行程事件行

## 部署

项目通过 Netlify 部署，配置在 `netlify.toml`：

```toml
[[redirects]]
  from = "/"
  to = "/invitation.html"
  status = 200
```

根路径 `/` 会重定向到邀请函页面。

## 注意事项

1. **移动端适配**：两个页面均已适配 375px+ 宽度，修改动效时请注意测试移动端显示
2. **localStorage**：页面使用 localStorage 存储倒计时状态（`48h_launched`、`48h_sim_time`），测试时可能需要清除
3. **Google Fonts**：使用 Bebas Neue + Barlow Condensed + Noto Sans SC 字体，需联网加载
4. **SVG 山峦**：页面底部的山峦轮廓是纯 SVG，可以调整路径或颜色
